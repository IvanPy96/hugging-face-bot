### Quick Start

#### Docker (recommended)

```bash
git clone <repo_url> && cd hf_bot
cp .env.example .env      # fill in BOT_TOKEN and CHAT_ID
docker compose up -d --build
```

```bash
docker compose logs -f     # logs
docker compose down        # stop
```

#### Local

```bash
python -m venv .venv && source .venv/bin/activate
pip install .
cp .env.example .env       # fill in BOT_TOKEN and CHAT_ID
python -m hf_bot
```
